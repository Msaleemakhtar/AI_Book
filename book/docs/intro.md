# Getting Started

Welcome to the **Physical AI & Humanoid Robotics** textbook!

This comprehensive 13-week course will teach you to control humanoid robots using:

- **ROS 2**: The robotic operating system
- **NVIDIA Isaac Sim**: GPU-accelerated simulation
- **Vision-Language-Action Models**: AI-powered robot control

## Course Structure

This textbook is organized into **4 modules** covering **13 weeks** of material:

### Module 1: ROS 2 - The Robotic Nervous System (Weeks 1-5)
Learn ROS 2 fundamentals, from basic concepts to URDF modeling and coordinate transformations.

### Module 2: Simulation Environments (Weeks 6-7)
Master Gazebo and Unity for robotics simulation before moving to Isaac Sim.

### Module 3: NVIDIA Isaac Platform (Weeks 8-10)
Explore Isaac Sim, Isaac ROS, and manipulation tasks with GPU acceleration.

### Module 4: Vision-Language-Action Models (Weeks 11-13)
Discover how humanoid robots learn from vision and language using VLA models.
